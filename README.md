# mori

どうぶつの森（Animal Crossing）の魚・虫の出現情報（季節・月・時間帯）と、ユーザーが実際に獲得したアイテム（捕獲済み）の管理を行う軽量なウェブアプリです。

## できること（概要）
- 各種（魚／虫）の出現月・出現時間帯を確認できる
- 北半球／南半球の切り替え表示に対応
- 「捕獲済み」をチェックして自分のコレクションを管理
- 月別・時間帯別フィルタで目的の種を絞り込める
- 状態の永続化（ブラウザのローカルストレージ、場合によってはサーバー保存）

## 主な機能
- 一覧表示：種名、出現月（北/南）、出現時間、出現場所など
- 捕獲管理：捕獲済みのチェック／解除、フィルタやソート
- フィルター：月／時間帯／カテゴリ（魚/虫）／半球
- データのインポート・エクスポート（未実装の場合は追加可能）

## ローカルでの確認（例）
1. リポジトリをクローン
   git clone https://github.com/misoclub/mori.git
2. ディレクトリへ移動
   cd mori
3. 依存関係をインストール（プロジェクトに合わせて）
   npm install
4. 開発サーバーを起動（package.json の scripts を参照）
   npm run dev  または  npm start

> 実際のコマンドは package.json を確認してください。

## データについて
- 出現情報（どの月・時間帯に出るか）はゲーム情報に基づく JSON/CSV 等で管理できます。data/ 以下に置く想定です。
- 捕獲済み情報はローカルストレージへ保存する仕組みが一般的ですが、ユーザー認証を追加すればサーバー保存にも対応可能です。
