# mori

どうぶつの森（Animal Crossing）の魚・虫の出現情報（季節・月・時間帯）と、ユーザーが実際に獲得したアイテムの管理を行うシンプルなウェブアプリです。

現在の README が別プロジェクトのままになっているため、実態に合わせて書き直しました。

## できること

- 各種（魚・虫）の出現月・出現時間帯を表示
- 北半球 / 南半球の切り替え表示
- 「捕獲済み」チェックで自分のコレクションを管理
- 月別・時間帯別のフィルタで探しやすく表示
- (場合によって) ブラウザのローカルストレージに状態を保存

## 技術スタック

- フロントエンド: JavaScript（このリポジトリは主に JavaScript で構成されています）
- マークアップ: HTML
- スタイル: CSS

## ローカルで動かす（例）

1. リポジトリをクローン

   git clone https://github.com/misoclub/mori.git

2. 依存関係をインストール

   npm install

3. 開発サーバーを起動

   npm run dev

※ 実際のコマンドは package.json の scripts を確認してください。

## データについて

- 出現情報（どの月・時間帯にどの虫や魚が出るか）はゲーム情報に基づいています。データが未整備であれば、CSV や JSON を追加して管理できます。
- 捕獲済みデータの永続化はローカルストレージまたはサーバーサイド保存に対応できます（実装状況に依存）。
